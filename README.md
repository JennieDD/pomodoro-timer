# 🍅 番茄钟 Pomodoro Timer

基于 **Python + PyQt5** 的 macOS 桌面番茄钟应用，界面简洁美观。

## 功能特性

- ⏱ **圆形进度环** — 实时显示剩余时间比例
- 🎨 **颜色主题** — 专注=红色、短休息=绿色、长休息=蓝色
- 🍅 **番茄计数** — 自动统计今日完成数量
- 🔔 **系统通知** — 阶段结束时弹出 macOS 原生通知
- 📌 **系统托盘** — 关闭窗口后最小化到菜单栏，双击恢复
- 🔄 **自动切换** — 每 4 个番茄后自动进入 15 分钟长休息

## 时间设置

| 阶段 | 时长 |
|------|------|
| 专注 | 25 分钟 |
| 短休息 | 5 分钟 |
| 长休息 | 15 分钟 |

## 安装与运行

```bash
# 安装依赖
pip3 install PyQt5

# 运行
python3 main.py
```
